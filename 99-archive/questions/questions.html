The ACF has created this archive so that interested schools or players can download and use these questions free of charge. The questions are in RTF format, stored in PC ZIP files. Save each tournament as a file and use a program like WinZip or Stuffit Expander to extract the rounds.

If you would like to donate ACF tournaments which are not already on this site, please send them to dhamiltn@wam.umd.edu.
